Master the scalar value of square matrices — from 2×2 expansions to Cramer's Rule for solving systems of linear equations.

📐 What is a Determinant?
A determinant is a particular type of expression written in a special concise form of equal rows and columns. It is defined only for square matrices.

2nd Order (2×2)

For matrix A = [a, b; c, d]
|A| = ad − bc
Example: |2, 4; −1, 2| = 4 + 4 = 8

3rd Order (3×3)

Expand along any row or column using cofactors.
Δ = a₁(b₂c₃−b₃c₂) − b₁(a₂c₃−a₃c₂) + c₁(a₂b₃−a₃b₂)

Order of Determinant

The number of rows (= number of columns) is called the order of the determinant.

Key Note

Only square matrices have determinants. Determinant is a scalar — a single number, not a matrix.
💡

Important: A determinant is a special case of a matrix. While a matrix is just an array, a determinant evaluates to a single scalar value. Notation: det(A) or |A|.

🧮 Minors and Cofactors

Minor (Mᵢⱼ)

The determinant obtained by deleting the i-th row and j-th column of a determinant is called the Minor of element aᵢⱼ.
Minor of aᵢⱼ = Mᵢⱼ
M₁₁ = a₂₂a₃₃ − a₂₃a₃₂
M₁₂ = a₂₁a₃₃ − a₂₃a₃₁

Cofactor (Cᵢⱼ)

Cofactor of element aᵢⱼ is defined as:
Cᵢⱼ = (−1)^(i+j) × Mᵢⱼ
If i+j = even → Cᵢⱼ = Mᵢⱼ
If i+j = odd → Cᵢⱼ = −Mᵢⱼ
⚠️

Sign pattern for cofactors: + − + / − + − / + − +. The top-left (1,1) is always +ve.

Value of Determinant (using cofactor expansion):
Δ = a₁₁A₁₁ + a₁₂A₁₂ + a₁₃A₁₃ (along R₁)
Δ = a₁₁A₁₁ + a₂₁A₂₁ + a₃₁A₃₁ (along C₁)
💜 Key Fact — Cross-Row/Column Rule

If elements of a row (or column) are multiplied with cofactors of any other row (or column), then their sum is always zero. This is frequently used in proofs.

🔗 Special Determinants

Symmetric Det.

|a h g; h b f; g f c| = abc + 2fgh − af² − bg² − ch²

Skew-Symmetric

Determinant of odd-order skew-symmetric matrix is always zero.
|0 a b; −a 0 c; −b −c 0| = 0

Cyclic Det. (Vandermonde)

|1 1 1; a b c; a² b² c²| = (a−b)(b−c)(c−a)

Max Value

For 3×3 matrix with entries ∈ {−1, 0, 1}, maximum |det| = 4
🔢 Expansion of Determinants

2×2 Determinant

|a b; c d| = ad − bc
Multiply main diagonal, subtract anti-diagonal

3×3 Expansion (R₁)

= a₁(b₂c₃−b₃c₂) − b₁(a₂c₃−a₃c₂) + c₁(a₂b₃−a₃b₂)

Sarrus' Rule (3×3)

a₁b₂c₃ + b₁c₂a₃ + c₁a₂b₃ − a₃b₂c₁ − b₃c₂a₁ − c₃a₂b₁

Cofactor Expansion (Any Row/Col)

Δ = Σ aᵢⱼ · Cᵢⱼ (sum over any row or col)
📐 Cyclic / Vandermonde Determinants

Type A

|1 1 1; a b c; a² b² c²| = (a−b)(b−c)(c−a)

Type B

|1 1 1; a b c; a³ b³ c³| = (a−b)(b−c)(c−a)(a+b+c)

Type C

|1 1 1; a² b² c²; a³ b³ c³| = (a−b)(b−c)(c−a)(ab+bc+ca)

Cyclic Pattern (a,b,c)

|a b c; b c a; c a b| = −(a³+b³+c³−3abc)
= −(a+b+c)(a²+b²+c²−ab−bc−ca) ≤ 0 for positive a,b,c
🎯

Cyclic Trick: When you see (a−b), (b−c), (c−a) as factors, use the cyclic determinant identities above. These appear in 80% of board proof questions.

🔗 Product of Two Determinants
Three Multiplication Methods:

Row × Column

Element (i,j) of product = dot product of i-th row of Δ₁ with j-th column of Δ₂

Row × Row

Element (i,j) of product = dot product of i-th row of Δ₁ with j-th row of Δ₂

Column × Column

Element (i,j) of product = dot product of i-th column of Δ₁ with j-th column of Δ₂
💡

Key Result: If Aᵢⱼ are cofactors of aᵢⱼ in Δ ≠ 0, then |Cofactor matrix| = Δ²

⚙️ 8 Properties of Determinants

Property 1 — Transpose

Value of determinant remains unchanged if rows and columns are interchanged.
|A| = |Aᵀ|
Note: If D' = −D → Skew Symmetric. Skew symmetric det. of odd order = 0.

Property 2 — Row/Column Interchange

If any two rows (or columns) are interchanged, the determinant changes sign only.
R₁ ↔ R₂ → Δ becomes −Δ

Property 3 — Identical Rows/Columns

If any two rows (or columns) are identical, then the value of determinant is zero.

Property 4 — Scalar Multiple

If each element of a row (or column) is multiplied by constant k, the determinant gets multiplied by k.
kR₁ → Δ' = kΔ
⚠️ For an n×n matrix: |kA| = kⁿ|A|

Property 5 — Sum of Elements

If elements of a row (or column) can be expressed as sum of two terms, the determinant splits into sum of two determinants.
|a₁+x b₁+y c₁+z; ...| = |a₁ b₁ c₁; ...| + |x y z; ...|

Property 6 — Row Operation

Value remains same if we add equimultiples of another row/column.
Rᵢ → Rᵢ + kRⱼ (i ≠ j)
Cᵢ → Cᵢ + kCⱼ (i ≠ j)
At least one row/column must remain unchanged!

Property 7 — Summation / Integration

If one row has function entries f₁(r), f₂(r), f₃(r) and rest are constants, then:
Σ Δᵣ = |Σf₁(r) Σf₂(r) Σf₃(r); constants; constants|

Property 8 — Factor Theorem

If substituting x = a makes Δ = 0, then (x − a) is a factor of Δ.

🔍 Proof: Vandermonde Determinant (Type A)
Prove: |1 1 1; a b c; a² b² c²| = (a−b)(b−c)(c−a)
Step-by-Step Proof
1
Apply C₁ → C₁ − C₂ and C₂ → C₂ − C₃ to get zeros in row 1.
2
Factor (a−b) from C₁ and (b−c) from C₂.
3
Expand along R₁ (top row has zeros except last entry).
4
Result: (a−b)(b−c){b+c − a−b} = (a−b)(b−c)(c−a) ✓
🔍 Proof: (a+b+c)³ Determinant
Prove: |a−b−c 2a 2a; 2b b−c−a 2b; 2c 2c c−a−b| = (a+b+c)³
Step-by-Step Proof
1
Apply C₁ → C₁ − C₂, C₂ → C₂ − C₃. Factor out −(a+b+c) from C₁ and (a+b+c) from C₂.
2
This gives (a+b+c)² × a 3×3 determinant with first row [−1, 0, 2a].
3
Expand along R₁: (a+b+c)²[−1(−c+a+b−2b) + 2a(1)]
4
Simplify: (a+b+c)² × (a+b+c) = (a+b+c)³ ✓
🔍 Proof: Cyclic Determinant = 0
Prove: |a b c; b c a; c a b| = −(a³+b³+c³−3abc) ≤ 0
Step-by-Step Proof
1
Apply C₁ → C₁ + C₂ + C₃ to get (a+b+c) in every entry of C₁.
2
Factor out (a+b+c) from C₁: get (a+b+c) × |1 b c; 1 c a; 1 a b|
3
Expand about C₁: (a+b+c){(cb−a²)−(b²−ac)+(ab−c²)}
4
= −(a+b+c)(a²+b²+c²−ab−bc−ca) = −(a³+b³+c³−3abc) ✓
🔍 Proof: 2abc(a+b+c)³
Prove: |(b+c)² ba ac; ba (c+a)² cb; ca cb (a+b)²| = 2abc(a+b+c)³
Key Steps
1
Apply C₁ → C₁/a, C₂ → C₂/b, C₃ → C₃/c (multiply abc outside).
2
Then R₁ → aR₁, R₂ → bR₂, R₃ → cR₃ to restore elements.
3
Apply C₁ → C₁−C₂ and C₂ → C₂−C₃ to create zeros.
4
Factor (a+b+c)² from C₁, C₂; then apply C₁ → aC₁+C₃, C₂ → bC₂+C₃.
5
Expand about R₃: gives 2abc(a+b+c)³ ✓
📊 System of Linear Equations
For a system: a₁x + b₁y + c₁z = d₁ , a₂x + b₂y + c₂z = d₂ , a₃x + b₃y + c₃z = d₃
Cramer's Rule Solution:
x = D₁/D , y = D₂/D , z = D₃/D   (provided D ≠ 0)
Where:
D (Coefficient Matrix)D₁ (Replace col 1 with constants)D₂ (Replace col 2)D₃ (Replace col 3)
|a₁ b₁ c₁; a₂ b₂ c₂; a₃ b₃ c₃| |d₁ b₁ c₁; d₂ b₂ c₂; d₃ b₃ c₃| |a₁ d₁ c₁; a₂ d₂ c₂; a₃ d₃ c₃| |a₁ b₁ d₁; a₂ b₂ d₂; a₃ b₃ d₃|

Geometric Interpretation (2 variables): Unique solution = intersecting lines; No solution = parallel lines; Infinite solutions = identical (coincident) lines.

🏠 Homogeneous System
When d₁ = d₂ = d₃ = 0 (all RHS = 0):

D ≠ 0

System has only trivial solution: x = y = z = 0. Exactly 1 solution.

D = 0

System has non-trivial solutions (in addition to trivial). Infinite solutions exist.
💜 Trivial Solution Definition

A system is said to have a trivial solution when all variables = 0 (i.e., x = y = z = 0). Non-trivial means at least one variable ≠ 0.

⭐ Standard Board & JEE Questions
Q1. Evaluate |x x+1; x−1 x|
Hint: Expand as x(x) − (x−1)(x+1) = x² − (x²−1)
Answer: 1 (constant, independent of x)
Q2. Without expanding, prove: |0 b −c; −b 0 a; c −a 0| = 0
Hint: The matrix is skew-symmetric of odd order (3). Property: det of odd-order skew-symmetric = 0.
Answer: D' = −D (by transposing) → D = −D → D = 0 ✓
Q3. Find x if |3 x; x 1| = |3 2; 4 1|
Hint: 3 − x² = 3 − 8, so x² = 8
Answer: x = ±2√2
Q4. Solve by Cramer's Rule: x+y+z=6, x−y+z=2, 3x+2y−4z=−5
Hint: Compute D=14, D₁=14, D₂=28, D₃=42
Answer: x = D₁/D = 1, y = D₂/D = 2, z = D₃/D = 3
Q5. For what values of α does the system αx+y+z=α−1, x+αy+z=α−1, x+y+αz=α−1 have infinite solutions?
Hint: D = (α−1)²(α+2). For infinite solutions, need D=0 AND equations consistent.
Answer: α = 1 only (α=−2 leads to no solution — contradictory equations)
Q6. If |A²| = 25 for A = |5 5α α; 0 α 5α; 0 0 5|, find |α|.
Hint: |A| = 25α (triangular matrix product of diagonals). |A²| = |A|² = 25.
Answer: (25α)² = 25 → 25α² = 1 → |α| = 1/5
Q7. Number of values of k for which 4x+ky+2z=0, kx+4y+z=0, 2x+2y+z=0 has non-zero solution.
Hint: For non-trivial: D=0 → k²−6k+8=0 → (k−2)(k−4)=0
Answer: 2 values (k = 2 and k = 4)
Q8. If a,b,c are real and x=cy+bz, y=az+cx, z=bx+ay (not all zero), find a²+b²+c²+2abc.
Hint: Non-trivial solution → D=0 → 1(1−a²)+c(−c−ab)−b(ac+b)=0
Answer: 1
Q9. Prove that |a−b−c 2a 2a; 2b b−c−a 2b; 2c 2c c−a−b| = (a+b+c)³
Hint: Apply C₁−C₂, C₂−C₃, factor (a+b+c)², then expand about R₁.
Answer: Proof completes to (a+b+c)² × (a+b+c) = (a+b+c)³ ✓
Q10. Find λ, μ for which x+y+z=6, x+2y+3z=10, x+2y+λz=μ has infinite solutions.
Hint: D=(λ−3). For D=0: λ=3. Then check D₁=0 requires μ=10.
Answer: λ = 5 and μ = 8 (unique infinite solution case)
💡 Key Facts to Remember
💡

Only square matrices have determinants. A 3×4 matrix has no determinant.

⚠️

|kA| ≠ k|A| for an n×n matrix. Correct: |kA| = kⁿ|A|. This is a very common exam trap!

🎯

Cramer's Rule only works when D ≠ 0. If D=0, you need to check each Dᵢ to classify no-solution vs. infinite solutions.

💡

For a skew-symmetric matrix of odd order, determinant is always 0. This is provable using D = D' = −D → D = 0.

🔢

The cofactor matrix determinant equals Δ² (for a 3×3): |cofactor matrix| = Δ². Useful in competition problems.

📐

Area of triangle with vertices (x₁,y₁), (x₂,y₂), (x₃,y₃) = ½|x₁(y₂−y₃) + x₂(y₃−y₁) + x₃(y₁−y₂)| = ½ × determinant form.

🎯

Maximum determinant of a 3×3 matrix with entries from {−1, 0, 1} is 4 (not 6 — 6 is impossible by contradiction).

🧠 Exam Strategy Tips
💜 Tip 1 — Expand Along the Sparsest Row/Column

Always expand along the row or column with the most zeros. This minimizes calculations. Use R₁ operation first to create zeros.

💜 Tip 2 — R₁+R₂+R₃ Trick

If all rows have the same sum, apply R₁ → R₁+R₂+R₃ first. This often creates a common factor (a+b+c) you can pull out.

💜 Tip 3 — Cramer's Rule Shortcut

For 2×2 systems: |a b; c d| → x = (d·b₁ − b·b₂)/D, y = (a·b₂ − c·b₁)/D where D = ad−bc. Memorize this pattern!

💜 Tip 4 — Cyclic Determinant Recognition

When you see symmetric patterns in a determinant (like a,b,c cycling), immediately think of Vandermonde identities. Check if (a−b)(b−c)(c−a) is a factor.

💜 Tip 5 — Consistency Check Order

For system analysis: First compute D. If D≠0 → unique solution (stop). If D=0 → compute D₁, D₂, D₃. All zero = infinite; any nonzero = no solution.

💜 Tip 6 — "Without Expanding" Questions

These always require using Property 6 (row operations) to simplify to a known pattern or create a zero row/column. Never actually expand — use properties!
