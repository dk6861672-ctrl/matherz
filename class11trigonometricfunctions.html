📏 Angle Measurement
An angle is formed by rotating a ray from its initial position to a terminal position. The amount of rotation defines the angle.

Degree Measure

One full rotation = 360°
Right angle = 90°
Straight angle = 180°

Radian Measure

Angle (in radians) = arc length ÷ radius
θ = l / r
One full rotation = 2π rad

Conversion Formula

1° = π/180 rad
1 rad = 180°/π ≈ 57.3°

Common Conversions

30° = π/6  |  45° = π/4
60° = π/3  |  90° = π/2
180° = π  |  270° = 3π/2
💡

Radians are preferred in higher mathematics because arc length = rθ is clean — no conversion factor needed.

📐 The Six Trigonometric Functions
Defined using a right-angled triangle: for an angle θ, consider the opposite (P), adjacent (B), and hypotenuse (H) sides.

sin θ

sin θ = P / H
Opposite ÷ Hypotenuse

cos θ

cos θ = B / H
Adjacent ÷ Hypotenuse

tan θ

tan θ = P / B = sin θ / cos θ
Opposite ÷ Adjacent

cosec θ

cosec θ = H / P = 1/sin θ
Reciprocal of sin

sec θ

sec θ = H / B = 1/cos θ
Reciprocal of cos

cot θ

cot θ = B / P = cos θ / sin θ
Reciprocal of tan
💜 Memory Trick — SOH CAH TOA

Sin = Opposite/Hypotenuse  ·  Cos = Adjacent/Hypotenuse  ·  Tan = Opposite/Adjacent

💜 Sin Value Trick — √0/2 to √4/2

sin values at 0°, 30°, 45°, 60°, 90° = √0/2, √1/2, √2/2, √3/2, √4/2 = 0, 1/2, 1/√2, √3/2, 1. Cos values are the reverse!

🧭 Signs in the Four Quadrants (ASTC Rule)
Trig functions are positive only in certain quadrants. The rule is A S T C — going anticlockwise from Quadrant I.

Quadrant I (0° – 90°)

All functions are positive
sin +, cos +, tan +

Quadrant II (90° – 180°)

Sin (and cosec) is positive
sin +, cos −, tan −

Quadrant III (180° – 270°)

Tan (and cot) is positive
sin −, cos −, tan +

Quadrant IV (270° – 360°)

Cos (and sec) is positive
sin −, cos +, tan −
💜 Mnemonic — "All Students Take Calculus"

All · Sin · Tan · Cos — one word for each quadrant in order (I → II → III → IV).

🔗 Pythagorean Identities

Identity 1

sin²θ + cos²θ = 1
→ sin²θ = 1 − cos²θ
→ cos²θ = 1 − sin²θ

Identity 2

1 + tan²θ = sec²θ
→ sec²θ − tan²θ = 1

Identity 3

1 + cot²θ = cosec²θ
→ cosec²θ − cot²θ = 1
➕ Sum & Difference Formulas

sin(A ± B)

sin(A+B) = sinA cosB + cosA sinB
sin(A−B) = sinA cosB − cosA sinB

cos(A ± B)

cos(A+B) = cosA cosB − sinA sinB
cos(A−B) = cosA cosB + sinA sinB

tan(A ± B)

tan(A+B) = (tanA + tanB) / (1 − tanA tanB)
tan(A−B) = (tanA − tanB) / (1 + tanA tanB)

Key Tip

For sin: mixed signs match (sin·cos ± cos·sin)
For cos: signs are opposite (cos·cos ∓ sin·sin)
✖️ Double Angle Formulas

sin 2A

sin 2A = 2 sinA cosA

cos 2A

cos 2A = cos²A − sin²A
= 2cos²A − 1
= 1 − 2sin²A

tan 2A

tan 2A = 2tanA / (1 − tan²A)
🔄 Sum-to-Product & Product-to-Sum

Sum → Product

sinC + sinD = 2sin((C+D)/2)cos((C−D)/2)
sinC − sinD = 2cos((C+D)/2)sin((C−D)/2)
cosC + cosD = 2cos((C+D)/2)cos((C−D)/2)
cosC − cosD = −2sin((C+D)/2)sin((C−D)/2)

Product → Sum

2sinA cosB = sin(A+B) + sin(A−B)
2cosA sinB = sin(A+B) − sin(A−B)
2cosA cosB = cos(A−B) + cos(A+B)
2sinA sinB = cos(A−B) − cos(A+B)

💜 Rule for Allied Angles

If the angle is a multiple of 90° (odd multiple), sin ↔ cos swap. If it's a multiple of 180° (even multiple), the function stays the same. Then apply the quadrant sign.

🔍 Derivation — sin(A+B) Formula
Geometric Proof using Unit Circle
1
Consider two angles A and B. Draw angle A from the x-axis, then add angle B from there.
2
Using the distance formula between two points on the unit circle, expand the squared distance.
3
From the expansion: 2 − 2cos(A−B) = 2 − 2(cosA cosB + sinA sinB)
4
This gives: cos(A−B) = cosA cosB + sinA sinB
5
Replace B with −B: cos(A+B) = cosA cosB − sinA sinB ✓
6
Use co-function identity (sin θ = cos(π/2−θ)) to derive sin(A+B) = sinA cosB + cosA sinB ✓
🔍 Derivation — Pythagorean Identity
Proof from Unit Circle Definition
1
For any angle θ, the point on the unit circle is (cos θ, sin θ).
2
The unit circle equation is: x² + y² = 1
3
Substituting x = cos θ and y = sin θ:
4
cos²θ + sin²θ = 1 — proved! ✓
5
Divide by cos²θ: 1 + tan²θ = sec²θ ✓
6
Divide by sin²θ: cot²θ + 1 = cosec²θ ✓
🔍 Derivation — Double Angle Formula (cos 2A)
Using cos(A+B) with A = B
1
Start with: cos(A+B) = cosA cosB − sinA sinB
2
Set B = A: cos(A+A) = cosA · cosA − sinA · sinA
3
Simplify: cos 2A = cos²A − sin²A ✓
4
Use sin²A = 1 − cos²A: cos 2A = 2cos²A − 1 ✓
5
Use cos²A = 1 − sin²A: cos 2A = 1 − 2sin²A ✓
⭐ Board & Exam Questions
Q1. Prove that sin²(π/6) + cos²(π/3) + tan²(π/4) = 2
Hint: Substitute standard values: sin(π/6)=1/2, cos(π/3)=1/2, tan(π/4)=1
Answer: (1/2)² + (1/2)² + 1² = 1/4 + 1/4 + 1 = 2 ✓
Q2. Find the value of sin 75°
Hint: Write 75° = 45° + 30°, then apply sin(A+B) formula
Answer: sin75° = sin45°cos30° + cos45°sin30° = (1/√2)(√3/2) + (1/√2)(1/2) = (√3+1)/(2√2)
Q3. Prove: (sin A + sin B) / (cos A + cos B) = tan((A+B)/2)
Hint: Use Sum-to-Product formulas on both numerator and denominator
Answer: Numerator = 2sin((A+B)/2)cos((A−B)/2); Denominator = 2cos((A+B)/2)cos((A−B)/2); Ratio = tan((A+B)/2) ✓
Q4. If tan A = 3/4 and A is in third quadrant, find sin A and cos A.
Hint: Use 1 + tan²A = sec²A to find cos A, then sin A = tan A · cos A. Apply Q3 signs.
Answer: sec²A = 1 + 9/16 = 25/16 → cos A = −4/5 (Q3, cos −ve); sin A = −3/5 (Q3, sin −ve)
Q5. Find the general solution of: sin θ = 1/2
Hint: Principal value is π/6. General solution for sin θ = sin α is θ = nπ + (−1)ⁿ α
Answer: θ = nπ + (−1)ⁿ (π/6), where n ∈ ℤ
Q6. Prove: cos 2x = (1 − tan²x) / (1 + tan²x)
Hint: Start from cos 2x = cos²x − sin²x, divide numerator & denominator by cos²x
Answer: (cos²x − sin²x)/cos²x ÷ (cos²x + sin²x)/cos²x = (1 − tan²x)/(1 + tan²x) ✓
Q7. Find the value of cos(−1710°)
Hint: Reduce the angle: 1710 = 4×360 + 270, so 1710° = 270° (coterminal). Then apply negative angle rule.
Answer: cos(−1710°) = cos(1710°) = cos(270°) = 0
💡 Key Facts to Remember
💡

Unit Circle is Everything: All six trig functions are simply coordinates and ratios of a point (cos θ, sin θ) on a circle of radius 1. Once you visualise this, the signs and values become obvious.

⚠️

Common Mistake — cos(A+B) ≠ cosA + cosB: Students often forget to apply the full expansion formula. cos(A+B) = cosA cosB − sinA sinB, not cosA + cosB.

🎯

Exam Favourite: Proving identities using sum-to-product formulas, and finding values like sin 75°, cos 15°, tan 105° appear in nearly every board paper.

📌

Periodicity: sin and cos repeat every 2π. tan and cot repeat every π. This means sin(θ + 2nπ) = sin θ for any integer n.

⚠️

tan 90° is Undefined — not infinity: At 90°, cos 90° = 0, so tan = sin/cos involves division by zero — the function is simply undefined at that point.

🧠 Exam Strategy Tips
💜 Tip 1 — Proving Identities

Always start from the more complex side and simplify to the simpler side. Convert everything to sin and cos first if stuck. Never cross-multiply both sides.

💜 Tip 2 — Finding Exact Values

If asked for sin 105°, write it as sin(60°+45°) or sin(180°−75°). Always break into standard angles (30°, 45°, 60°, 90°).

💜 Tip 3 — General Solutions

Learn all three forms: sin θ = sin α → θ = nπ + (−1)ⁿα, cos θ = cos α → θ = 2nπ ± α, tan θ = tan α → θ = nπ + α. These are frequently asked 3-mark questions.

💜 Tip 4 — Allied Angles (90°±θ rule)

For 90° ± θ and 270° ± θ: sin ↔ cos swap. For 180° ± θ and 360° ± θ: function stays the same. Then just fix the sign using ASTC.

💜 Tip 5 — Maximum Marks on Proofs

Write the LHS and RHS steps clearly and end with "= RHS" or "= LHS". Examiners award step marks, so even partial working earns marks.
