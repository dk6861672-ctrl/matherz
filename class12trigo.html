🔄 What Are Inverse Trig Functions?
Inverse trigonometric functions are the inverse of the trigonometric functions — they give back the angle when given a ratio.
But since sin, cos, tan etc. are not one-one on their full domain, we restrict their domain to make them bijective — only then can an inverse exist.

Notation

sin⁻¹x is NOT 1/sin x
sin⁻¹x means "the angle whose sine is x"
Also written as: arcsin x

Why Restrict Domain?

sin x = 1/2 has infinitely many solutions: 30°, 150°, 390°...
We restrict to [−π/2, π/2] so sin⁻¹(1/2) = π/6 uniquely.

💚 Key Observation

sin⁻¹ and tan⁻¹ have range around 0 (negative and positive), while cos⁻¹ and cot⁻¹ have range in [0, π]. This matters for sign determination.

🔗 Complementary Angle Properties

sin⁻¹ + cos⁻¹

sin⁻¹x + cos⁻¹x = π/2
Valid for x ∈ [−1, 1]

tan⁻¹ + cot⁻¹

tan⁻¹x + cot⁻¹x = π/2
Valid for x ∈ ℝ

sec⁻¹ + cosec⁻¹

sec⁻¹x + cosec⁻¹x = π/2
Valid for |x| ≥ 1
🔄 Negative Argument Properties

Odd Functions (negative goes out)

sin⁻¹(−x) = −sin⁻¹x, x ∈ [−1,1]
tan⁻¹(−x) = −tan⁻¹x, x ∈ ℝ
cosec⁻¹(−x) = −cosec⁻¹x, |x|≥1

Even-like (π minus rule)

cos⁻¹(−x) = π − cos⁻¹x, x ∈ [−1,1]
sec⁻¹(−x) = π − sec⁻¹x, |x|≥1
cot⁻¹(−x) = π − cot⁻¹x, x ∈ ℝ
🔗 Reciprocal Identities for Inverse Functions

sin⁻¹ ↔ cosec⁻¹

sin⁻¹(1/x) = cosec⁻¹x
for |x| ≥ 1

cos⁻¹ ↔ sec⁻¹

cos⁻¹(1/x) = sec⁻¹x
for |x| ≥ 1

tan⁻¹ ↔ cot⁻¹

tan⁻¹(1/x) = cot⁻¹x, x > 0
tan⁻¹(1/x) = −π + cot⁻¹x, x < 0
➕ Addition Formulas for tan⁻¹

tan⁻¹x + tan⁻¹y

If xy < 1: = tan⁻¹((x+y)/(1−xy))
If xy > 1, x > 0: = π + tan⁻¹((x+y)/(1−xy))
If xy > 1, x < 0: = −π + tan⁻¹((x+y)/(1−xy))

tan⁻¹x − tan⁻¹y

If xy > −1: = tan⁻¹((x−y)/(1+xy))
If xy < −1, x > 0: = π + tan⁻¹((x−y)/(1+xy))
🔢 Double & Triple Angle in Inverse Form

Double Angle Inverse Formulas

2sin⁻¹x = sin⁻¹(2x√(1−x²)), |x| ≤ 1/√2
2cos⁻¹x = cos⁻¹(2x²−1), 0 ≤ x ≤ 1
2tan⁻¹x = tan⁻¹(2x/(1−x²)), |x| < 1
2tan⁻¹x = sin⁻¹(2x/(1+x²)), |x| ≤ 1
2tan⁻¹x = cos⁻¹((1−x²)/(1+x²)), x ≥ 0

Compositions (Simplifications)

tan(tan⁻¹x) = x, x ∈ ℝ
sin⁻¹(sin x) = x, x ∈ [−π/2, π/2]
cos⁻¹(cos x) = x, x ∈ [0, π]
tan⁻¹(tan x) = x, x ∈ (−π/2, π/2)

sin⁻¹x = cos⁻¹(√(1−x²))

cos⁻¹ in terms of others

cos⁻¹x = sin⁻¹(√(1−x²))

🔍 Derivation — sin⁻¹x + cos⁻¹x = π/2
Proof
1
Let sin⁻¹x = θ, where θ ∈ [−π/2, π/2] and x ∈ [−1,1]
2
Then sin θ = x
3
We know: sin θ = cos(π/2 − θ)
4
So: x = cos(π/2 − θ), which means cos⁻¹x = π/2 − θ
5
Therefore: θ + cos⁻¹x = π/2 → sin⁻¹x + cos⁻¹x = π/2 ✓
🔍 Derivation — tan⁻¹x + tan⁻¹y = tan⁻¹((x+y)/(1−xy))
Proof (for xy < 1)
1
Let tan⁻¹x = α and tan⁻¹y = β, so tan α = x, tan β = y
2
Use addition formula: tan(α+β) = (tan α + tan β)/(1 − tan α tan β)
3
Substitute: tan(α+β) = (x + y)/(1 − xy)
4
So: α + β = tan⁻¹((x+y)/(1−xy))
5
Therefore: tan⁻¹x + tan⁻¹y = tan⁻¹((x+y)/(1−xy)) ✓ when xy < 1
🔍 Derivation — 2tan⁻¹x = sin⁻¹(2x/(1+x²))
Proof for |x| ≤ 1
1
Let tan⁻¹x = θ, so x = tan θ and θ ∈ (−π/2, π/2)
2
sin 2θ = 2 tan θ/(1 + tan²θ) = 2x/(1 + x²)
3
So: 2θ = sin⁻¹(2x/(1+x²))
4
Therefore: 2tan⁻¹x = sin⁻¹(2x/(1+x²)) ✓
🔍 Derivation — sin⁻¹(−x) = −sin⁻¹x
Proof
1
Let sin⁻¹x = θ, so sin θ = x and θ ∈ [−π/2, π/2]
2
sin(−θ) = −sin θ = −x
3
So: sin⁻¹(−x) = −θ = −sin⁻¹x
4
Therefore: sin⁻¹(−x) = −sin⁻¹x ✓
⭐ CBSE Board-Level Questions
Q1. Find the value of sin⁻¹(sin 3π/5)
Hint: sin⁻¹(sin x) = x only if x ∈ [−π/2, π/2]. Here 3π/5 ∉ [−π/2, π/2].
sin(3π/5) = sin(π − 3π/5) = sin(2π/5). Since 2π/5 ∈ [−π/2, π/2], answer = 2π/5
Q2. Prove: 3sin⁻¹x = sin⁻¹(3x − 4x³)
Hint: Let sin⁻¹x = θ, so x = sin θ. Then sin 3θ = 3sinθ − 4sin³θ = 3x − 4x³. Hence 3θ = sin⁻¹(3x − 4x³).
Verified using triple angle formula: sin 3θ = 3sinθ − 4sin³θ
Q3. Find the value of tan⁻¹(1) + tan⁻¹(1/√3)
tan⁻¹(1) = π/4, tan⁻¹(1/√3) = π/6. Check: product = 1/√3 < 1, so use addition formula.
= tan⁻¹((1 + 1/√3)/(1 − 1/√3)) = tan⁻¹((√3+1)/(√3−1)) = tan⁻¹(tan(5π/12)) = 5π/12
Q4. Write the function f(x) = tan⁻¹(√((1−cosx)/(1+cosx))) in simplified form.
Use half-angle: (1−cosx)/(1+cosx) = tan²(x/2). So √ = |tan(x/2)|.
f(x) = x/2, for x ∈ (0, π)
Q5. If sin⁻¹x + sin⁻¹y = π/2, find cos⁻¹x + cos⁻¹y.
Use sin⁻¹x = π/2 − cos⁻¹x and sin⁻¹y = π/2 − cos⁻¹y.
cos⁻¹x + cos⁻¹y = π − (sin⁻¹x + sin⁻¹y) = π − π/2 = π/2
Q6. Solve: tan⁻¹2x + tan⁻¹3x = π/4
Apply addition formula: tan⁻¹((2x+3x)/(1−6x²)) = π/4. So 5x/(1−6x²) = 1.
6x² + 5x − 1 = 0 → (6x − 1)(x + 1) = 0 → x = 1/6 (x = −1 rejected since it gives negative)
Q7. Simplify: tan⁻¹(cosx/(1+sinx)), x ∈ (−π/2, π/2)
Divide numerator and denominator by cos(x/2). Use half-angle identities.
= tan⁻¹(tan(π/4 − x/2)) = π/4 − x/2
💡 Key Facts About Inverse Trig
📐

The principal value is the unique angle in the restricted range. Always check if the given angle is in the principal value branch before applying sin⁻¹(sin x) = x.

🔄

Inverse trig functions are used in integration (Class 12 calculus): ∫ 1/√(1−x²) dx = sin⁻¹x + C, ∫ 1/(1+x²) dx = tan⁻¹x + C

📊

The graphs of y = sin⁻¹x and y = sin x are reflections of each other in the line y = x (that's what inverse functions do geometrically).

🎯

tan⁻¹x has horizontal asymptotes at y = ±π/2. This means tan⁻¹(∞) = π/2 and tan⁻¹(−∞) = −π/2 — very useful in limits and integration.

⚡

In navigation and physics, inverse trig functions calculate angles from ratios — e.g., finding the angle of a slope, the direction of a vector, etc.

🧠 Board Exam Strategies
💚 Tip 1 — Always Check the Domain

Before applying any inverse trig formula, verify the argument is in the correct domain. sin⁻¹ needs x ∈ [−1,1], tan⁻¹ needs the product condition checked.

💚 Tip 2 — The tan⁻¹ Addition Formula Trap

tan⁻¹x + tan⁻¹y has THREE cases based on the sign of (1 − xy) and the quadrant. For boards, xy < 1 case (simple formula) is most common. Always state the condition!

💚 Tip 3 — Simplification Strategy

When asked to simplify expressions like tan⁻¹(√((1−cosx)/(1+cosx))), always try half-angle substitutions. Most Class 12 problems reduce to x/2 or π/4 ± x/2.

💚 Tip 4 — Principal Value Problems

For sin⁻¹(sin 7π/6): First find sin(7π/6) = −1/2. Then sin⁻¹(−1/2) = −π/6. Don't blindly write 7π/6!
