🔗 What is a Relation?
Definition

If A and B are two non-empty sets, then any subset R of A×B is called a Relation from set A to set B.

Notation

R : A → B  means  R ⊆ A×B
If (x,y) ∈ R → write xRy (x is R-related to y)
If (x,y) ∉ R → write x R̸ y (x is NOT R-related to y)

Domain & Range

Domain — set of all first coordinates of R
Dom(R) = {a : (a,b) ∈ R for some b ∈ B}
Range — set of all second coordinates of R
Range(R) = {b : (a,b) ∈ R for some a ∈ A}
💡 Key Note

A relation R on set A means the relation from A to A itself, i.e. R ⊆ A×A. The co-domain is the full set B, but Range ⊆ Co-domain.

🗂️ Types of Relations

Empty Relation

R in set A is empty if no element of A is related to any element of A.
R = φ ⊂ A×A

Universal Relation

R in set A where each element is related to every element of A.
R = A×A

Identity Relation

R = {(x,y) : x∈A, y∈A, x=y}
OR  R = {(x,x) : x∈A}
Every element maps only to itself.

Reflexive Relation

If (a,a) ∈ R for every a∈A
Every element is related to itself.
Note: Identity ⊆ Reflexive (Identity is a special reflexive relation)

Symmetric Relation

If (a₁,a₂) ∈ R ⟹ (a₂,a₁) ∈ R
for all a₁,a₂ ∈ A
aRb ⟹ bRa

Transitive Relation

If (a₁,a₂)∈R and (a₂,a₃)∈R
⟹ (a₁,a₃) ∈ R  for all a₁,a₂,a₃∈A
aRb and bRc ⟹ aRc

Equivalence Relation

R is equivalence if it is:
✅ Reflexive  ✅ Symmetric  ✅ Transitive
Example: "is equal to" (=) on ℤ is an equivalence relation.

Antisymmetric Relation

R is antisymmetric if:
(a,b)∈R and (b,a)∈R ⟹ a=b
∀ a,b ∈ R
Example: "≤" on ℝ is antisymmetric.
🔁 Inverse Relation
Definition

If A and B are two non-empty sets and R be a relation from A to B such that R = {(a,b) : a∈A, b∈B}, then the inverse of R, denoted by R⁻¹, is a relation from B to A defined by:

R⁻¹ = {(b,a) : (a,b) ∈ R}

Property 1

(R⁻¹)⁻¹ = R

Property 2

Dom(R⁻¹) = Range(R)

Property 3

Range(R⁻¹) = Dom(R)
🎯 Equivalence Class
Definition

Let R be an equivalence relation on a non-empty set A. For all a∈A, the equivalence class of 'a' is defined as the set of all such elements of A which are related to 'a' under R. It is denoted by [a].

[a] = {x∈A : (x,a) ∈ R}
💡

Two equivalence classes are either identical or disjoint. The collection of all equivalence classes forms a partition of the set A.

📐 What is a Function?
Definition

Let X and Y be two non-empty sets. Then a rule f which associates to each element x∈X, a unique element, denoted by f(x) of Y, is called a function from X to Y, written as f : X → Y

Image

f(x) is called the image of x under f

Pre-image

x is called the pre-image of f(x)

Range

f(X) = {f(x) : x∈X}
Range ⊆ Co-domain (Y)
⚡ Critical Condition for a Function

Every element of the domain must have exactly one image in the co-domain. If any element maps to two values, it is NOT a function. If any domain element has no image, it is NOT a function.

🗂️ Equal Functions
Definition

Two functions f and g are called equal functions if they are defined on the same domain A and f(a) = g(a) for every a∈A. Then f = g.

1️⃣ One-One (Injective) Function
Definition

A function f: X→Y is one-one (injective) if distinct elements of X have distinct images in Y.

x₁,x₂∈X : f(x₁)=f(x₂) ⟹ x₁=x₂

Equivalently (Contrapositive)

x₁ ≠ x₂ ⟹ f(x₁) ≠ f(x₂)

Graphical Test

Any horizontal line cuts the graph at most once ⟹ one-one.
💡 Tips

A strictly increasing OR strictly decreasing function is always one-one. If f'(x) > 0 or f'(x) < 0 for all x in domain, it's one-one.

🔢 Many-One Function
Definition

f: A→B is many-one if two or more elements of A have the same f-image in B. i.e., x₁,x₂∈A, x₁≠x₂ but f(x₁) = f(x₂).

📌

Any continuous function which has at least one local maximum or minimum is many-one. If a horizontal line cuts the graph at two or more points, f is many-one.

⚡

One-One + Many-One = Total number of mappings. If a function is one-one, it cannot be many-one and vice versa.

2️⃣ Onto (Surjective) Function
Definition

f: A→B is onto (surjective) if every element of B (co-domain) is the image of at least one element of A. i.e., Range = Co-domain.

∀ b∈B, ∃ a∈A such that f(a) = b

💡 Key Notes

If range = co-domain, f(x) is onto. Any polynomial of odd degree, f: ℝ→ℝ, is onto. Polynomial of even degree defined from ℝ→ℝ is always into (not onto).

📦 Into Function
Definition

f: A→B is into if there exists at least one element in the co-domain B which is not the image of any element in the domain. i.e., Range ⊊ Co-domain.

📌

A function is either onto OR into — never both. If a function is onto, it cannot be into and vice versa.

🏆 Bijective Function (One-One + Onto)
Definition

f: A→B is bijective if it is both injective (one-one) and surjective (onto). Bijective functions are also called invertible, non-singular, or biuniform functions.

Condition

n(A) = n(B) is necessary for bijection between finite sets.
Number of bijections = n! where n = n(A) = n(B)

Why Important?

A function f: A→B has an inverse function f⁻¹: B→A if and only if f is bijective.

🔄 Composition of Functions
Definition

Let f: A→B and g: B→C be two functions. Then the function gof: A→C defined by (gof)(x) = g(f(x)) ∀ x∈A is called the composite of the two functions f and g.

x ─f→ f(x) ─g→ g(f(x)) = (gof)(x)

NOT Commutative

gof ≠ fog  in general
Order matters! g(f(x)) ≠ f(g(x)) in general.

IS Associative

fo(goh) = (fog)oh
If f,g,h are three functions such that both fo(goh) and (fog)oh are defined.
🧩 Properties of Composite Functions
① The composite of two one-one functions is one-one.
② The composite of two onto functions is onto.
③ The composite of two bijections is a bijection.
④ If gof is one-one then f is one-one (but g need not be).
⑤ If gof is onto then g is onto (but f need not be).
⑥ If gof is bijective, f is injective and g is surjective.
🔙 Inverse of a Function
Definition

Let f: A→B be a bijective function. Then the function f⁻¹: B→A defined by f⁻¹(b) = a ⟺ f(a) = b is called the inverse function of f.

Key Result

fof⁻¹ = I_B
f⁻¹of = I_A
Where I is the identity function.

Properties

(f⁻¹)⁻¹ = f
(gof)⁻¹ = f⁻¹og⁻¹
Note: Inverse exists only for bijective functions!
📐 How to Find f⁻¹(x)

Step 1: Let y = f(x).   Step 2: Express x in terms of y.   Step 3: Replace y with x to get f⁻¹(x).   Step 4: Verify that f⁻¹(f(x)) = x.

📐 Composition of Relations
Definition

Let R ⊆ A×B and S ⊆ B×C be two relations. Then the composition SoR ⊆ A×C is defined by: (a,c) ∈ SoR if ∃ b∈B such that (a,b)∈R and (b,c)∈S

⊕ Binary Operations
Definition

A binary operation * on a set A is a function *: A×A → A. So * associates each ordered pair (a,b) of elements of A to a unique element a*b in A. This is called closure property.

Commutative

* is commutative if a*b = b*a ∀ a,b∈A
Example: + and × on ℝ are commutative. − is not.

Associative

* is associative if (a*b)*c = a*(b*c) ∀ a,b,c∈A
Example: + and × are associative. − and ÷ are not.

Identity Element

e is identity if a*e = e*a = a ∀ a∈A
Identity for +: 0. Identity for ×: 1.
Identity element is unique if it exists.

Inverse Element

b is inverse of a if a*b = b*a = e
Inverse of a under +: −a. Under ×: 1/a (a≠0).
Inverse is unique for each element.

Distributive

* distributes over # if:
a*(b#c) = (a*b)#(a*c)
Example: × distributes over + on ℝ.

Closure Property

* is closed on A if a*b ∈ A ∀ a,b∈A
Example: ℤ is closed under + but not under ÷.

⭐ CBSE Board-Level Questions
Q1. If A = {1,2,3,4,5}, write the relation R = {(a,b): |a−b| is even} and check if it's an equivalence relation.
Hint: R = {(1,3),(1,5),(3,5),(2,4),(1,1),(2,2),...}. Check reflexive (a−a=0, even ✅), symmetric (|a−b|=|b−a| ✅), transitive (|a−b| even, |b−c| even ⟹ |a−c| even ✅).
Yes, it is an equivalence relation. Equivalence classes: [1]={1,3,5}, [2]={2,4}.
Q2. Show that f: ℝ→ℝ defined by f(x) = 2x−1 is a bijection and find f⁻¹(x).
One-one: f(x₁)=f(x₂) ⟹ 2x₁−1=2x₂−1 ⟹ x₁=x₂ ✅. Onto: for any y∈ℝ, x=(y+1)/2 ∈ℝ ✅.
f⁻¹(x) = (x+1)/2
Q3. If f: {1,3} → {1,2,5} and g: {1,2,5} → {1,2,3,4} be given by f = {(1,2),(3,5)} and g = {(1,3),(2,3),(5,1)}. Write down gof.
gof(1) = g(f(1)) = g(2) = 3. gof(3) = g(f(3)) = g(5) = 1.
gof = {(1,3),(3,1)}
Q4. Let g, f: ℝ→ℝ be defined by g(x) = (x+2)/3 and f(x) = 3x−2. Write fog.
fog(x) = f(g(x)) = f((x+2)/3) = 3·((x+2)/3) − 2.
fog(x) = (x+2) − 2 = x. So fog = Iᴿ (identity function).
Q5. If f: ℝ→ℝ defined by f(x) = (2x−1)/5 is invertible, find f⁻¹(x).
Let y = (2x−1)/5 ⟹ 5y = 2x−1 ⟹ x = (5y+1)/2.
f⁻¹(x) = (5x+1)/2
Q6. If f(x) = x/(x+1), x≠−1. Find fof(x).
fof(x) = f(f(x)) = f(x/(x+1)). Substitute into the formula.
fof(x) = [x/(x+1)] / [x/(x+1)+1] = x/(2x+1), x≠−½
Q7. Let * be a binary operation on R defined by a*b = a+b+ab. Find 3*4. Is * commutative and associative?
3*4 = 3+4+(3)(4) = 7+12 = 19. For commutative: a*b = a+b+ab = b+a+ba = b*a ✅. For associative: (a*b)*c = (a+b+ab)*c = a+b+ab+c+(a+b+ab)c.
3*4 = 19. Yes, * is both commutative and associative.
Q8. Is the relation R = {(1,1),(2,2),(3,3),(1,2),(2,1),(2,3),(1,3)} on A={1,2,3} an equivalence relation?
Check reflexive: (1,1),(2,2),(3,3) all present ✅. Symmetric: (1,2)∈R and (2,1)∈R ✅, (2,3)∈R but check (3,2) — NOT present ❌.
Not an equivalence relation. It is NOT symmetric (since (2,3)∈R but (3,2)∉R). Also not transitive.
Q9. (2 mark) Check if R on ℕ defined by R = {(a,b): a = b²} is reflexive, symmetric or transitive.
Reflexive: (a,a)∈R requires a=a², i.e. a=0 or a=1 only, not all a∈ℕ. Symmetric: (4,2)∈R but (2,4)∉R since 2≠16. Transitive: (16,4)∈R and (4,2)∈R but (16,2)∉R since 16≠4.
R is neither reflexive, nor symmetric, nor transitive.
