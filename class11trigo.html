📊 All 4 Quadrants — ASTC Rule

Quadrant I (0 to π/2)

ALL ratios positive: sin, cos, tan, cosec, sec, cot

Quadrant II (π/2 to π)

Only sin & cosec positive
cos, tan, sec, cot → negative

Quadrant III (π to 3π/2)

Only tan & cot positive
sin, cos, sec, cosec → negative

Quadrant IV (3π/2 to 2π)

Only cos & sec positive
sin, tan, cosec, cot → negative

💜 Rule for Allied Angles

🔗 Product-to-Sum & Sum-to-Product

Product → Sum

2sinA cosB = sin(A+B) + sin(A−B)
2cosA sinB = sin(A+B) − sin(A−B)
2cosA cosB = cos(A−B) + cos(A+B)
2sinA sinB = cos(A−B) − cos(A+B)

Sum → Product

sinC+sinD = 2sin((C+D)/2)cos((C−D)/2)
sinC−sinD = 2cos((C+D)/2)sin((C−D)/2)
cosC+cosD = 2cos((C+D)/2)cos((C−D)/2)
cosC−cosD = −2sin((C+D)/2)sin((C−D)/2)

🔍 Derivation — sin(A+B) = sinA cosB + cosA sinB
Geometric Proof
1
Consider a unit circle. Place angle A, then angle B additional rotation.
2
The point P at angle (A+B) has coordinates (cos(A+B), sin(A+B)).
3
Using the rotation matrix approach: rotating point (cosB, sinB) by angle A:
x' = cosA·cosB − sinA·sinB = cos(A+B)
y' = sinA·cosB + cosA·sinB = sin(A+B)
4
Therefore: sin(A+B) = sinA cosB + cosA sinB ✓
🔍 Derivation — Double Angle: sin 2A
From Addition Formula
1
Start with: sin(A+B) = sinA cosB + cosA sinB
2
Put B = A: sin(A+A) = sinA cosA + cosA sinA
3
Therefore: sin 2A = 2 sinA cosA ✓
🔍 Derivation — cos 2A (3 forms)
From cos(A+A)
1
cos(A+A) = cosA cosA − sinA sinA = cos²A − sin²A
2
Since sin²A = 1 − cos²A: = cos²A − (1−cos²A) = 2cos²A − 1
3
Since cos²A = 1 − sin²A: = (1−sin²A) − sin²A = 1 − 2sin²A
4
All 3 forms: cos 2A = cos²A − sin²A = 2cos²A − 1 = 1 − 2sin²A ✓
🔍 Derivation — Half-Angle Formulas
From cos 2A = 1 − 2sin²A
1
Put A = θ/2: cos θ = 1 − 2sin²(θ/2)
2
Rearrange: sin²(θ/2) = (1−cosθ)/2
3
Therefore: sin(θ/2) = ±√((1−cosθ)/2) ✓
4
Similarly from cos θ = 2cos²(θ/2) − 1: cos(θ/2) = ±√((1+cosθ)/2) ✓
⭐ Important Questions for Exams
Q1. Prove that: sin(A+B)·sin(A−B) = sin²A − sin²B
Hint: Use sin(A+B) = sinA cosB + cosA sinB and multiply. Apply sin² + cos² = 1.
Q2. If sin x + sin y = a and cos x + cos y = b, find cos(x−y).
Hint: Square both equations, add them, use sum-to-product identities. Answer: cos(x−y) = (a²+b²−2)/2
Q3. Prove: cos²x + cos²(x+π/3) + cos²(x−π/3) = 3/2
Hint: Expand using cos(x+π/3) and cos(x−π/3) formulas, use cos2θ = 2cos²θ−1, simplify.
Q4. Find the value of sin 75° and cos 75°
Use sin(45°+30°). sin 75° = (√6+√2)/4, cos 75° = (√6−√2)/4
Q5. Prove: (sin 3x + sin x) sin x + (cos 3x − cos x) cos x = 0
Use sum-to-product formulas. sin 3x + sin x = 2sin 2x cos x, cos 3x − cos x = −2sin 2x sin x. Substitute and simplify.
Q6. Find the principal solution of tan x = −√3
tan(π/3) = √3, so tan(π − π/3) = −tan(π/3) = −√3. Principal solutions: x = 2π/3 and x = 5π/3
Q7. If A + B + C = π, prove sin 2A + sin 2B + sin 2C = 4 sinA sinB sinC
Classic triangle identity. Use sum-to-product on first two terms, then use C = π−A−B.
💡 Key Facts & Interesting Points
🌍

The word trigonometry was first used by Bartholomaeus Pitiscus in 1595. But the concepts existed in ancient Egypt and Babylon.

🔭

Astronomers use trig to calculate distances to stars. sin(1 arcsecond) ≈ 1/206265 — this is used to define a parsec, a unit of interstellar distance.

📐

For acute angles: sin is increasing and cos is decreasing. At 45°, both equal 1/√2 — the exact midpoint.

∞

The general solution of sin x = k is x = nπ + (−1)ⁿ sin⁻¹k, not just two values. There are infinitely many solutions!

🎵

Sound waves, light waves, AC electricity — all described by y = A sin(ωt + φ). Trigonometry literally governs wave physics.

🧠 Exam Tips for Class 11
💜 Tip 1 — Allied Angles: The Name Rule

For 90°±θ and 270°±θ: function changes (sin↔cos, tan↔cot, sec↔cosec). For 180°±θ and 360°±θ: function stays same. Then determine sign from quadrant.

💜 Tip 2 — General Solutions

sin x = sin α → x = nπ + (−1)ⁿα
cos x = cos α → x = 2nπ ± α
tan x = tan α → x = nπ + α

💜 Tip 3 — Maximum & Minimum Values

For a sinθ + b cosθ: Max = √(a²+b²), Min = −√(a²+b²)
This is because: a sinθ + b cosθ = √(a²+b²) sin(θ+φ) where tan φ = b/a

💜 Tip 4 — Proving Identities Strategy

1. Convert to sin and cos
2. Find common denominators
3. Use Pythagorean identities (sin²+cos²=1)
4. Factor strategically
5. Never move a term from one side to another!
