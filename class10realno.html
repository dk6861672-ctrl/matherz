🔢 The Real Number System
Real numbers (ℝ) include every number on the number line — both rational and irrational. Understanding their structure is the foundation of this chapter.

Natural Numbers (ℕ)

Counting numbers starting from 1.
ℕ = {1, 2, 3, 4, ...}
Every natural number is also a whole, integer, rational, and real number.

Whole Numbers (W)

Natural numbers plus zero.
W = {0, 1, 2, 3, ...}
0 is whole but NOT a natural number.

Integers (ℤ)

All whole numbers and their negatives.
ℤ = {..., −2, −1, 0, 1, 2, ...}

Rational Numbers (ℚ)

Numbers expressible as p/q where p, q ∈ ℤ, q ≠ 0.
e.g. 1/2, −3/4, 7, 0.333...
Decimal expansion is either terminating or recurring.

Irrational Numbers

Cannot be written as p/q. Non-terminating, non-repeating decimals.
e.g. √2, √3, π, e

Real Numbers (ℝ)

Union of all rational and irrational numbers. Covers entire number line.
ℝ = ℚ ∪ (Irrationals)
💡

Key Hierarchy: ℕ ⊂ W ⊂ ℤ ⊂ ℚ ⊂ ℝ. Every set is a proper subset of the next. Irrational numbers are NOT a subset of ℚ — they form a completely separate part of ℝ.

➗ Euclid's Division Lemma
The cornerstone theorem of this chapter — it formalises the concept of division with remainder.

The Lemma (Theorem)

For any two positive integers a and b, there exist unique integers q (quotient) and r (remainder) such that:
a = bq + r, where 0 ≤ r < b
Here: a = dividend, b = divisor, q = quotient, r = remainder.
📌

Example: For a = 17, b = 5 → 17 = 5 × 3 + 2. Here q = 3, r = 2. Note r = 2 satisfies 0 ≤ 2 < 5. ✓

💜 Uniqueness Matters

The lemma guarantees the values of q and r are unique — there's exactly one way to write a = bq + r satisfying 0 ≤ r < b. This uniqueness is what makes the algorithm work.

🔄 Euclid's Division Algorithm (for HCF)
The algorithm applies the Division Lemma repeatedly to find the HCF (Highest Common Factor) of two positive integers.

Algorithm Steps

Step 1 Apply Lemma: write a = bq₁ + r₁ (if r₁ = 0, HCF = b)
Step 2 Now apply lemma to (b, r₁): write b = r₁·q₂ + r₂
Step 3 Continue: r₁ = r₂·q₃ + r₃, and so on...
Stop When remainder = 0. The last non-zero remainder is the HCF.
🧮

Worked Example — HCF(135, 225):
225 = 135 × 1 + 90
135 = 90 × 1 + 45
90 = 45 × 2 + 0 → HCF = 45

🧮

Worked Example — HCF(196, 38220):
38220 = 196 × 195 + 0 → HCF = 196

🏛️ Fundamental Theorem of Arithmetic
One of the most powerful theorems in mathematics — every composite number has a unique prime factorisation.

The Theorem

Every composite number can be expressed (factorised) as a product of primes, and this factorisation is unique (except for the order of factors).
n = p₁ᵃ¹ × p₂ᵃ² × p₃ᵃ³ × ...
where p₁, p₂, p₃,... are distinct primes and a₁, a₂, a₃,... are positive integers.

HCF via Prime Factorisation

HCF = Product of the smallest powers of each common prime factor.
HCF(12, 18) = 2¹ × 3¹ = 6

LCM via Prime Factorisation

LCM = Product of the highest powers of each prime factor (common or not).
LCM(12, 18) = 2² × 3² = 36
⭐

Golden Relationship: For any two positive integers a and b — HCF(a,b) × LCM(a,b) = a × b. This is one of the most commonly tested results in CBSE exams!

√ Irrational Numbers
Numbers that cannot be expressed in the form p/q where p, q are integers and q ≠ 0. Their decimal expansions are non-terminating and non-repeating.

Common Irrationals

√2 = 1.41421356...
√3 = 1.73205080...
√5 = 2.23606797...
π = 3.14159265...

Key Properties

Sum/difference of a rational and irrational is irrational.
Product of a non-zero rational and irrational is irrational.
√p is irrational if p is a prime number.
⚠️

Common Confusion: √4 = 2 is rational (since 2 = 2/1). Only √p where p is prime is irrational. Always simplify the square root first before concluding it's irrational!

💜 Quick Check for Terminating Decimal

Express p/q in lowest terms first. Then factorise q. If q = 2ⁿ × 5ᵐ (and nothing else), the decimal terminates. Otherwise, it repeats. This is a favourite 1-mark question in CBSE!

📌

Worked Example: Is 13/3125 terminating? 3125 = 5⁵. Since q = 5⁵ = 2⁰ × 5⁵, it terminates. 13/3125 = 0.00416.

📌 Euclid's Division Lemma

Core Formula

a = bq + r
Where a > b > 0, q ≥ 0, and 0 ≤ r < b
a = dividend, b = divisor, q = quotient, r = remainder
🏛️ Fundamental Theorem of Arithmetic

Prime Factorisation

n = p₁ᵃ¹ · p₂ᵃ² · p₃ᵃ³ · ...
Each pᵢ is a distinct prime; factorisation is unique.

HCF × LCM Relation

HCF(a,b) × LCM(a,b) = a × b
Only valid for TWO numbers.

HCF Formula

HCF = product of least powers of common prime factors

LCM Formula

LCM = product of highest powers of all prime factors
🔁 Decimal Expansion Condition

Terminating Decimal

p/q terminates if q (in lowest terms) is of the form:
q = 2ⁿ × 5ᵐ
(n, m are non-negative integers)

Non-Terminating Recurring

p/q does NOT terminate if q (lowest terms) has a prime factor other than 2 or 5.

🔍 Proof — √2 is Irrational
Method: Proof by Contradiction (most important proof for board exams)
Proof by Contradiction
1
Assume √2 is rational. Then √2 = p/q where p, q are integers, q ≠ 0, and p/q is in lowest terms (i.e., HCF(p,q) = 1).
2
Squaring both sides: 2 = p²/q² → p² = 2q². So p² is even, which means p is even.
3
Since p is even, write p = 2m for some integer m. Substituting: (2m)² = 2q² → 4m² = 2q² → q² = 2m².
4
So q² is even, which means q is also even. Therefore both p and q are even.
5
Contradiction! If both p and q are even, then HCF(p,q) ≥ 2. But we assumed HCF(p,q) = 1. This is a contradiction.
6
Conclusion: Our assumption was wrong. Therefore √2 is irrational ✓
🔍 Proof — √3 is Irrational
Proof by Contradiction
1
Assume √3 = p/q in lowest terms (HCF(p,q) = 1).
2
Squaring: 3q² = p². So 3 divides p², and since 3 is prime, 3 divides p (by FTA).
3
Let p = 3c. Then: 3q² = 9c² → q² = 3c². So 3 divides q² → 3 divides q.
4
Both p and q are divisible by 3 → HCF(p,q) ≥ 3. Contradicts HCF = 1.
5
Therefore √3 is irrational ✓
💜 Generalisation

The same method proves √p is irrational for ANY prime p. The key step is: "if p divides a², then p divides a" — this only holds when p is prime. This is where the Fundamental Theorem of Arithmetic is used!

🔍 Proof — (5 − √3) is Irrational
Proof by Contradiction
1
Assume (5 − √3) is rational. Then (5 − √3) = p/q for some integers p, q with q ≠ 0.
2
Rearranging: √3 = 5 − p/q = (5q − p)/q.
3
Since p, q are integers, (5q − p)/q is rational. So √3 would be rational.
4
Contradiction! We proved √3 is irrational. Hence our assumption was wrong.
5
∴ (5 − √3) is irrational ✓
🎯

Pattern: To prove (a ± b√p) is irrational (where a is rational, p is prime): assume it's rational, isolate √p, show √p equals a rational number — contradiction!

🔍 Proof — Every Positive Even Integer is of the form 2q
Using Euclid's Division Lemma
1
Let n be any positive integer. By Euclid's lemma with b=2: n = 2q + r, where r = 0 or 1.
2
If r = 0: n = 2q → n is even. If r = 1: n = 2q+1 → n is odd.
3
Therefore every positive even integer is of form 2q and every odd integer is of form 2q+1 ✓
⭐ NCERT & Board Exam Questions
Q1. Use Euclid's Division Algorithm to find HCF(135, 225).
Hint: Apply the lemma repeatedly — start with 225 = 135q + r.
Answer: 225 = 135×1+90 → 135 = 90×1+45 → 90 = 45×2+0 ∴ HCF = 45
Q2. Express 140 as a product of its prime factors.
Hint: Divide by 2 first, then continue with other primes.
Answer: 140 = 2² × 5 × 7
Q3. Find LCM(26, 91) using prime factorisation. Also verify: HCF × LCM = product of numbers.
Hint: 26 = 2×13, 91 = 7×13. Common factor = 13.
Answer: HCF = 13, LCM = 182. Check: 13 × 182 = 2366 = 26 × 91 ✓
Q4. Prove that √5 is irrational.
Hint: Assume √5 = p/q (HCF=1), square both sides, show 5 | p and 5 | q → contradiction.
Answer: If √5 = p/q → 5q² = p² → 5|p → p=5c → q² = 5c² → 5|q. HCF≥5, contradicts HCF=1. ∴ √5 is irrational.
Q5. Without actually performing long division, state whether 13/3125 has a terminating or non-terminating decimal expansion.
Hint: Factorise 3125 and check if it's of the form 2ⁿ × 5ᵐ.
Answer: 3125 = 5⁵ = 2⁰ × 5⁵. Since denominator = 2ⁿ × 5ᵐ, it TERMINATES.
Q6. Check whether 6ⁿ can end with the digit 0 for any natural number n.
Hint: For a number to end in 0, it must have both 2 and 5 as prime factors.
Answer: 6ⁿ = (2×3)ⁿ = 2ⁿ × 3ⁿ. Since 5 is never a factor of 6ⁿ, it can NEVER end with 0.
Q7. Prove that 3 + 2√5 is irrational.
Hint: Assume rational, isolate √5, show it equals rational — contradiction.
Answer: If 3+2√5 = p/q → √5 = (p−3q)/2q = rational. But √5 is irrational. Contradiction. ∴ 3+2√5 is irrational.
Q8. Find the largest number that divides 245 and 1029 leaving remainder 5 in each case.
Hint: Subtract 5 from each number, then find HCF of the results.
Answer: HCF(245−5, 1029−5) = HCF(240, 1024) = 16. Largest number = 16.
Q9. An army contingent of 616 members is to march behind an army band of 32 members. Find the maximum number of columns such that both groups march in the same number.
Hint: This is an HCF problem — find HCF(616, 32).
Answer: 616 = 2³×7×11, 32 = 2⁵. HCF = 2³ = 8. Maximum columns = 8.
Q10. Show that any positive odd integer is of the form 4q+1 or 4q+3, where q is some integer.
Hint: Apply Euclid's lemma with b=4. Possible remainders are 0, 1, 2, 3. Eliminate even cases.
Answer: n = 4q+r, r∈{0,1,2,3}. r=0,2 gives even numbers (4q, 4q+2). For odd n, r must be 1 or 3 → n = 4q+1 or 4q+3. ✓
💡 Key Facts to Remember
💡

1 is neither prime nor composite. It has exactly one factor (itself). Always remember this for prime factorisation questions.

💡

2 is the only even prime number. All other even numbers have 2 as a factor, so they're composite.

⭐

HCF × LCM = Product of two numbers — this formula only works for exactly TWO numbers, not three. Don't apply it to three numbers!

⚠️

Common Mistake: √4 = 2 is RATIONAL (not irrational). Always simplify surds first — √4, √9, √16, √25 etc. are all rational.

⚠️

Common Mistake: When checking terminating decimals, first reduce p/q to lowest terms. Then check if denominator = 2ⁿ × 5ᵐ. For example, 6/15 = 2/5 → terminates (not 6/15 directly).

🎯

Exam Trend: Proof of irrationality (2–3 marks) is the most frequently asked question from this chapter. Practice proofs for √2, √3, √5 and composite forms like 3+2√5, 5−√3, 7√5.

📌

Euclid vs FTA: Euclid's algorithm is used to find HCF quickly without prime factorisation. But prime factorisation (FTA) is used to find BOTH HCF and LCM together. Choose wisely based on the question!

🧠 Exam Strategy Tips
💜 Tip 1 — Euclid's Algorithm Direction

Always start with the larger number as the dividend. Write: larger = smaller × q + r. Never start with the smaller one — it will give a quotient of 0 and waste a step.

💜 Tip 2 — Contradiction Proof Structure

For irrationality proofs, always follow this exact structure: (1) Assume rational = p/q in lowest terms, (2) Square/manipulate to get prime divides p², (3) Conclude prime divides p, (4) Substitute and show prime divides q, (5) State contradiction with HCF = 1, (6) Conclude. Don't skip steps in board exams!

💜 Tip 3 — The "Ending in 0" Type Questions

For questions like "Can 6ⁿ / 8ⁿ / 9ⁿ end in zero?", write out the prime factorisation. A number ends in 0 only if it has both 2 and 5 as factors. If 5 is absent from the factorisation, the answer is always NO.

💜 Tip 4 — HCF from Word Problems

Word problems asking for the "largest size", "maximum number of groups", "biggest equal share" are always HCF problems. Problems asking for "minimum time to meet again", "smallest common container" are always LCM problems.

💜 Tip 5 — Decimal Expansion Fast Method

For terminating decimals, multiply numerator and denominator to make denominator a power of 10. E.g., 3/8 = 3/(2³) = 3×5³/(2³×5³) = 375/1000 = 0.375. This also tells you how many decimal places: equal to max(n,m) where denominator = 2ⁿ × 5ᵐ.

💜 Tip 6 — Remainder Problems

If a number leaves remainder r when divided by d, subtract r from the number first, then find HCF. "Find the largest number that divides a and b leaving same remainder r" → Find HCF(a−r, b−r).
