Polynomials

Understand zeroes, their geometric meaning, and the deep relationship between zeroes and coefficients — from linear to cubic polynomials, with the Division Algorithm.

📌 What is a Polynomial?
A polynomial in one variable x is an expression of the form:
p(x) = aₙxⁿ + aₙ₋₁xⁿ⁻¹ + ... + a₁x + a₀
where aₙ, aₙ₋₁, ..., a₀ are real numbers (called coefficients), aₙ ≠ 0, and n is a non-negative integer.

Degree

Highest power of the variable in the polynomial.
degree of 3x² − 5x + 2 = 2

Leading Coefficient

Coefficient of the highest-degree term.
In 3x² − 5x + 2, it is 3

Constant Term

The term with no variable (x⁰ term).
In 3x² − 5x + 2, it is 2
⚠️

Not Polynomials: √x (fractional power), 1/x (negative power), 2x⁻¹ + 3. The variable's exponent must always be a non-negative integer.

💡

Key Rule: A polynomial of degree n can have at most n zeroes. This is not "exactly n" — a quadratic can have 0, 1, or 2 real zeroes.

🎯 Zeroes of a Polynomial
A zero (or root) of a polynomial p(x) is a value of x for which p(x) = 0. It is also called a root.

Finding Zeroes — Linear

For p(x) = ax + b, set p(x) = 0:
ax + b = 0 → x = −b/a
Linear polynomial has exactly 1 zero.

Finding Zeroes — Quadratic

For p(x) = ax² + bx + c, use factorisation or quadratic formula:
x = (−b ± √(b²−4ac)) / 2a

Geometric Meaning

The zeroes of p(x) are the x-coordinates where the graph y = p(x) intersects the x-axis.

Verifying a Zero

Substitute x = α into p(x). If p(α) = 0, then α is a zero.
E.g. p(x) = x² − 3x + 2, p(1) = 1 − 3 + 2 = 0 ✓
🔗 Relationship Between Zeroes and Coefficients — Quadratic
For a quadratic polynomial p(x) = ax² + bx + c, if α and β are the zeroes, then:

Sum of Zeroes (α + β)

α + β = −b/a
= −(coefficient of x) / (coefficient of x²)

Product of Zeroes (αβ)

αβ = c/a
= constant term / coefficient of x²

Reconstructing a Quadratic from its Zeroes

If α, β are zeroes of a quadratic, then the polynomial is:
p(x) = k[x² − (α+β)x + αβ]
where k is any non-zero real constant. Usually k = 1 for simplest form.
🧮

Example: p(x) = 2x² − 8x + 6. Here a=2, b=−8, c=6.
Sum of zeroes = −(−8)/2 = 4. Product of zeroes = 6/2 = 3.
Verify: zeroes are 1 and 3. Sum = 4 ✓, Product = 3 ✓

🔗 Relationship Between Zeroes and Coefficients — Cubic
For a cubic polynomial p(x) = ax³ + bx² + cx + d, if α, β, γ are the three zeroes:

Sum of Zeroes

α+β+γ = −b/a

Sum of Products (pairs)

αβ+βγ+γα = c/a

Product of all Zeroes

αβγ = −d/a
🧮

Example: p(x) = x³ − 4x² + 5x − 2. a=1, b=−4, c=5, d=−2.
Sum = −(−4)/1 = 4. Sum of products = 5/1 = 5. Product = −(−2)/1 = 2.
Zeroes are 1, 1, 2: Sum=4 ✓ Pairs = 1×1+1×2+1×2 = 5 ✓ Product = 2 ✓

➗ Division Algorithm for Polynomials
Just like integers, polynomials can be divided. If p(x) and g(x) are polynomials with g(x) ≠ 0, then:

Division Algorithm

p(x) = g(x) × q(x) + r(x)
where p(x) = dividend, g(x) = divisor, q(x) = quotient, r(x) = remainder.
The degree of r(x) < degree of g(x), or r(x) = 0.

Application 1

Find remaining zeroes of a polynomial when some zeroes are known — divide out the known factor and factorise the quotient.

Application 2

Find the value of unknowns when remainder is given — divide and equate coefficients of r(x) to the given remainder.
💜 Key Condition

For the Division Algorithm: degree of remainder r(x) must be strictly less than degree of divisor g(x). If g(x) is linear (degree 1), r(x) must be a constant (degree 0).

🔗 Zeroes–Coefficient Relations

Quadratic — p(x) = ax² + bx + c

α + β = −b/a  (Sum of zeroes)
α · β = c/a  (Product of zeroes)
α − β = √[(α+β)² − 4αβ] (Difference)

Cubic — p(x) = ax³ + bx² + cx + d

α+β+γ = −b/a
αβ+βγ+γα = c/a
α·β·γ = −d/a
📐 Constructing Polynomials from Zeroes

Quadratic from zeroes α, β

p(x) = k[x² − (α+β)x + αβ]
k = any non-zero constant (usually 1)

Cubic from zeroes α, β, γ

p(x) = k[x³ − (α+β+γ)x² + (αβ+βγ+γα)x − αβγ]
➗ Division Algorithm

Core Formula

Dividend = Divisor × Quotient + Remainder
p(x) = g(x) · q(x) + r(x)
Condition: deg r(x) < deg g(x) OR r(x) = 0
📊 Discriminant (Nature of Zeroes)
For quadratic ax² + bx + c, the discriminant D = b² − 4ac determines the nature of zeroes:
Discriminant DNature of ZeroesGraph Behaviour
D > 0Two distinct real zeroesParabola cuts x-axis at 2 points
D = 0Two equal real zeroes (repeated root)Parabola touches x-axis at 1 point
D < 0No real zeroesParabola does not touch x-axis
💜 Geometric Insight

The number of zeroes = number of times the graph cuts (not touches) the x-axis. When D = 0 the graph touches — this counts as one zero (repeated), NOT two separate zeroes.

🔢 Useful Algebraic Identities for Zeroes

Derived from α + β and αβ

α² + β² = (α+β)² − 2αβ
α² − β² = (α+β)(α−β)
α³ + β³ = (α+β)³ − 3αβ(α+β)
α³ − β³ = (α−β)³ + 3αβ(α−β)

Reciprocal Zeroes

1/α + 1/β = (α+β)/αβ = −b/c
1/(αβ) = a/c
If one zero is reciprocal of other: αβ = 1 → c = a

🔍 Reading Zeroes from a Graph

Rule

Zeroes = x-coordinates where the curve meets or crosses the x-axis (y = 0 line).
Count the number of intersections = number of real zeroes.

Touching vs Crossing

Crosses x-axis: zero has odd multiplicity (simple root).
Touches & bounces back: zero has even multiplicity (repeated root, D=0 for quadratic).
🎯

CBSE Exam Tip: If a graph is shown and you're asked "how many zeroes does p(x) have?" — just count the number of times the curve touches/crosses the x-axis, not the y-axis!

🔍 Derivation — Sum & Product of Zeroes (Quadratic)
If α and β are zeroes of ax² + bx + c, we can derive the relations from first principles.
Step-by-Step Derivation
1
Since α and β are zeroes: p(x) = a(x − α)(x − β) for some leading coefficient a.
2
Expand: a(x−α)(x−β) = a[x² − (α+β)x + αβ] = ax² − a(α+β)x + a(αβ)
3
Compare with ax² + bx + c:
→ Coefficient of x: −a(α+β) = b ⟹ α+β = −b/a
4
→ Constant term: a(αβ) = c ⟹ αβ = c/a
5
Therefore: α+β = −b/a    αβ = c/a ✓
🔍 Derivation — Zeroes–Coefficient Relations (Cubic)
Step-by-Step Derivation
1
If α, β, γ are zeroes of ax³+bx²+cx+d, write: p(x) = a(x−α)(x−β)(x−γ)
2
Expand step by step:
(x−α)(x−β) = x² − (α+β)x + αβ
3
Multiply by (x−γ):
= x³ − (α+β+γ)x² + (αβ+βγ+γα)x − αβγ
4
Multiply through by a and compare with ax³+bx²+cx+d:
−a(α+β+γ) = b ⟹ α+β+γ = −b/a
5
a(αβ+βγ+γα) = c ⟹ αβ+βγ+γα = c/a
6
−a(αβγ) = d ⟹ αβγ = −d/a ✓
🔍 Worked Example — Polynomial Long Division
Divide p(x) = x³ − 3x² − x + 3 by g(x) = x² − 4x + 3. Find quotient and remainder.
Step-by-Step Long Division
1
Divide leading term of dividend by leading term of divisor:
x³ ÷ x² = x. So first term of quotient = x.
2
Multiply: x × (x² − 4x + 3) = x³ − 4x² + 3x.
Subtract from dividend: (x³ − 3x² − x + 3) − (x³ − 4x² + 3x) = x² − 4x + 3
3
Divide leading term: x² ÷ x² = 1. Second term of quotient = 1.
4
Multiply: 1 × (x² − 4x + 3) = x² − 4x + 3.
Subtract: (x² − 4x + 3) − (x² − 4x + 3) = 0
5
Remainder = 0. Quotient = (x + 1). Verify: (x²−4x+3)(x+1) = x³−3x²−x+3 ✓
📌

Note: Since remainder = 0, (x²−4x+3) is a factor of (x³−3x²−x+3). This means the zeroes of x²−4x+3 (which are 1 and 3) are also zeroes of the cubic!

🔍 Worked Example — Finding Zeroes of a Cubic Given One Zero
Find all zeroes of p(x) = 2x³ + x² − 5x + 2, given that one zero is 1/2.
Step-by-Step Solution
1
Since 1/2 is a zero, (x − 1/2) is a factor, i.e. (2x − 1) is a factor.
2
Divide p(x) by (2x−1) using long division:
2x³ + x² − 5x + 2 ÷ (2x−1) = x² + x − 2 (remainder = 0)
3
Factorise the quotient: x² + x − 2 = (x+2)(x−1).
4
So p(x) = (2x−1)(x+2)(x−1). Setting each factor to 0:
x = 1/2, x = −2, x = 1
5
All three zeroes: 1/2, −2, 1 ✓
⭐ NCERT & Board Exam Questions
Q1. Find the zeroes of the polynomial p(x) = x² − 3 and verify the relationship between zeroes and coefficients.
Hint: Set x² − 3 = 0 and solve for x. Then check sum = −b/a and product = c/a.
Answer: Zeroes are √3 and −√3. Sum = √3 + (−√3) = 0 = −0/1 = −b/a ✓. Product = (√3)(−√3) = −3 = −3/1 = c/a ✓
Q2. Find a quadratic polynomial whose zeroes are 2 and −3.
Hint: Use p(x) = k[x² − (sum of zeroes)x + product of zeroes].
Answer: Sum = 2+(−3) = −1. Product = 2×(−3) = −6. p(x) = x² + x − 6.
Q3. Find the zeroes of p(x) = 6x² − 3 − 7x and verify the relation between zeroes and coefficients.
Hint: Rewrite as 6x² − 7x − 3, then factorise (split middle term).
Answer: 6x²−7x−3 = (2x−3)(3x+1). Zeroes: 3/2 and −1/3. Sum = 3/2−1/3 = 7/6 = −(−7)/6 ✓. Product = (3/2)(−1/3) = −1/2 = −3/6 ✓
Q4. Divide 2x² + 3x + 1 by x + 2. State the quotient and remainder.
Hint: Use polynomial long division. Degree of quotient = 2−1 = 1.
Answer: Quotient = 2x − 1. Remainder = 3. Verify: (x+2)(2x−1)+3 = 2x²+3x−2+3 = 2x²+3x+1 ✓
Q5. If α and β are zeroes of x² − 4x + 3, find the value of α² + β².
Hint: Use the identity α² + β² = (α+β)² − 2αβ.
Answer: α+β = 4, αβ = 3. α²+β² = 4² − 2(3) = 16 − 6 = 10.
Q6. Find all the zeroes of 2x⁴ − 3x³ − 3x² + 6x − 2, if two zeroes are √2 and −√2.
Hint: (x−√2)(x+√2) = x²−2 is a factor. Divide the polynomial by (x²−2) and factorise the quotient.
Answer: Divide by (x²−2): quotient = 2x²−3x+1 = (2x−1)(x−1). All zeroes: √2, −√2, 1/2, 1.
Q7. If the sum and product of zeroes of a quadratic polynomial are respectively 1/4 and −1, find the polynomial.
Hint: Use p(x) = k[x² − (sum)x + product]. Take k = 4 for integer coefficients.
Answer: p(x) = x² − (1/4)x − 1 or equivalently 4x² − x − 4.
Q8. Check whether x² + 3x + 1 is a factor of 3x⁴ + 5x³ − 7x² + 2x + 2.
Hint: Divide and check if remainder = 0.
Answer: Dividing 3x⁴+5x³−7x²+2x+2 by x²+3x+1 gives quotient 3x²−4x+2 and remainder 0. Yes, it is a factor. ✓
Q9. If α and β are zeroes of p(x) = ax² + bx + c, find the value of 1/α + 1/β in terms of a, b, c.
Hint: 1/α + 1/β = (α+β)/(αβ). Substitute known relations.
Answer: 1/α + 1/β = (α+β)/(αβ) = (−b/a)/(c/a) = −b/c.
Q10. If one zero of the quadratic polynomial x² + 3x + k is 2, find the value of k.
Hint: Substitute x = 2 in the polynomial and set p(2) = 0.
Answer: p(2) = 4 + 6 + k = 0 → k = −10.
💡 Key Facts to Remember
💡

Degree = Max Real Zeroes: A polynomial of degree n can have AT MOST n real zeroes. A quadratic CAN have 0, 1, or 2 zeroes — it's not always 2!

💡

Geometric Meaning of Zeroes: Zeroes of a polynomial are the x-coordinates of points where the graph y = p(x) intersects the x-axis. This is the single most tested concept in this chapter.

⭐

Reconstructing Polynomial: Given sum (S) and product (P) of zeroes of a quadratic: p(x) = k(x² − Sx + P). Take k = 1 unless coefficients need to be integers.

⚠️

Common Sign Error: Sum of zeroes = −b/a (negative b over a). Students often forget the negative sign. Remember: if p(x) = x² − 5x + 6, sum = −(−5)/1 = +5, not −5!

⚠️

Cubic Sign Error: Product of zeroes of cubic ax³+bx²+cx+d = −d/a (negative d over a). The middle relation αβ+βγ+γα = +c/a has a positive sign.

🎯

Exam Trend: "Find all zeroes of a polynomial given some zeroes" (3–4 marks) is asked almost every year. Practice dividing by factors corresponding to given zeroes.

📌

Graph questions: If a quadratic graph opens upward (∪) and is entirely above x-axis → D < 0 → no real zeroes. If it's a ∪ touching x-axis at one point → D = 0 → one repeated zero.

🧠 Exam Strategy Tips
💜 Tip 1 — Split the Middle Term (Factorisation)

For ax² + bx + c: find two numbers that multiply to a×c and add to b. Split bx into those two terms, then group and factorise. This is faster than the quadratic formula for CBSE questions.

💜 Tip 2 — Long Division Shortcut Check

After long division, always verify using: Divisor × Quotient + Remainder = Dividend. This single check catches all errors and can save you from losing all marks on a 3-mark question.

💜 Tip 3 — When Given "Two Zeroes of a Degree-4 Polynomial"

If zeroes are ±√k, form factor (x²−k). Divide the quartic by this factor → quotient is a quadratic. Factorise the quotient to get the remaining 2 zeroes. This pattern appears every year.

💜 Tip 4 — Reading Graphs (1–2 mark questions)

The number of times the graph cuts the x-axis = number of distinct real zeroes. Touches without crossing = repeated root (1 zero, not 2). Above x-axis entirely = 0 zeroes. Don't confuse x-intercepts with y-intercepts!

💜 Tip 5 — k Value in Polynomial Construction

When constructing a polynomial from given zeroes, k = 1 gives the simplest answer. But if the question says "find a polynomial with integer coefficients" and αβ is fractional, choose k to clear the denominator.

💜 Tip 6 — Verify Every Answer

After finding zeroes α, β, always verify: substitute back into p(x) to check p(α) = 0 and p(β) = 0. For relations, verify sum and product formulas. This habit prevents careless errors in board exams.
