📐 What is a Matrix?
A matrix is an ordered rectangular array of numbers (or functions) arranged in m rows and n columns, enclosed in brackets [  ] or (  ) or ‖ ‖. Denoted as A = [aᵢⱼ]ₘₓₙ.

Order & Size

A matrix with m rows and n columns has order m × n.
No. of elements = m × n
For square matrix (m = n):  elements = n²

Element Notation

aᵢⱼ = element in i-th row and j-th column.
Example: a₂₃ → 2nd row, 3rd column.
A = [aᵢⱼ]ₘₓₙ   i = 1..m, j = 1..n

Diagonal Elements

Elements a₁₁, a₂₂, a₃₃, …, aₙₙ are called Diagonal Elements.
The line along which they lie is called the Principal or Leading Diagonal.

Conjugate Elements

In a square matrix, aᵢⱼ and aⱼᵢ are called Conjugate Elements.
a₁₂ ↔ a₂₁   a₁₃ ↔ a₃₁   a₂₃ ↔ a₃₂
⚠️

Critical Rule: Determinant of a matrix exists ONLY for a square matrix. For any non-square matrix (m ≠ n), determinant is not defined.

🧠

In a square matrix n×n: Principal diagonal has n elements (i = j). Elements strictly above diagonal: (n²−n)/2. Same count below diagonal. Total: n² elements.

✅ Equality of Two Matrices
Two matrices A = [aᵢⱼ] and B = [bᵢⱼ] are equal if and only if both conditions hold:

Condition 1 — Same Order

They must have the same number of rows and columns (same order m × n).

Condition 2 — Equal Elements

Every corresponding element is equal: aᵢⱼ = bᵢⱼ for all i and j.
💜 Board Exam Tip

When A = B is given, set up individual element equations (a₁₁ = b₁₁, a₁₂ = b₁₂, etc.) and solve simultaneously. Do not try to guess values mentally.

🔡 All Types of Matrices

Row Matrix (Row Vector)

Only one row, any number of columns.
Order: 1 × n
A = [a₁₁  a₁₂  …  a₁ₙ]

Column Matrix (Column Vector)

Only one column, any number of rows.
Order: m × 1
A = [a₁₁; a₂₁; … aₘ₁]

Zero / Null Matrix (O)

All entries are zero. Denoted O or 0ₘₓₙ.
aᵢⱼ = 0 for all i, j

Horizontal Matrix

Order m × n with n > m (more cols than rows).
Example: [1 3 5 6 1; 2 3 5 7 1]₂₅

Vertical Matrix

Order m × n with m > n (more rows than cols).
Example: [3 4; 1 2; 2 5; 1 1]₄₂

Singleton Matrix

Only one element.
Order: 1 × 1

Square Matrix (Order n)

Number of rows = Number of columns.
m = n
Only square matrices have a defined diagonal, determinant, and can be symmetric/skew-symmetric.

Diagonal Matrix

A square matrix where all non-diagonal elements = 0.
aᵢⱼ = 0 if i ≠ j
Diagonal elements can be anything (including zero).

Scalar Matrix

A diagonal matrix where all diagonal elements are equal.
d₁ = d₂ = d₃ = a (const)
Special case of diagonal matrix.

Unit / Identity Matrix (I)

Scalar matrix where all diagonal elements = 1.
aᵢⱼ = 1 if i=j, else 0
Det(I) = |I| = 1

💜 Hierarchy Trick

Identity ⊂ Scalar ⊂ Diagonal ⊂ Square. Each is a more general form of the previous. A scalar matrix IS a diagonal matrix. An identity matrix IS a scalar matrix.

➕ Addition & Subtraction
Two matrices can be added/subtracted only if they have the same order.

Definition

If A = [aᵢⱼ] and B = [bᵢⱼ] are both m×n:
A + B = [aᵢⱼ + bᵢⱼ]
A − B = [aᵢⱼ − bᵢⱼ]

Properties of Addition

A + B = B + A (Commutative)
(A+B)+C = A+(B+C) (Associative)
A + O = A (Identity)
A + (−A) = O (Inverse)
🔢 Scalar Multiplication
If k is a scalar and A = [aᵢⱼ], then kA = [k · aᵢⱼ] — every element multiplied by k.

Properties

k(A+B) = kA + kB
(k+l)A = kA + lA
k(lA) = (kl)A
(−1)A = −A

Negative of Matrix

−A = (−1)A = [−aᵢⱼ]
Every element is negated.
A − B = A + (−B)
✖️ Matrix Multiplication (AB)
Condition: Columns of A = Rows of B. If A is m×n and B is n×p, then AB is m×p.
cᵢⱼ = ∑ₖ aᵢₖ · bₖⱼ — dot product of i-th row of A with j-th column of B.

Properties

A(BC) = (AB)C Associative
A(B+C) = AB + AC
AI = IA = A
AO = OA = O

⚠ NOT Commutative!

AB ≠ BA  (in general)
Even if both products are defined and of same order, they are generally different.

⚠ Zero Product Warning

AB = O  does NOT mean  A = O or B = O.
AB = AC  does NOT imply  B = C (even if A ≠ O).

Result Order Rule

A(m×n) × B(n×p) = C(m×p)
The inner n’s must match. Result takes the outer dimensions.
💜 Memory Trick — Matrix Multiply

For C = AB: element cᵢⱼ = (row i of A) · (column j of B). Each element of C is a dot product. Visualise "row scans left-to-right, column scans top-to-bottom."

🔄 Transpose of a Matrix
The transpose of A (written A' or Aᵀ) is obtained by interchanging rows and columns. If A = [aᵢⱼ]ₘₓₙ, then A' = [aⱼᵢ]ₙₓₘ.

Properties of Transpose

(A')' = A
(kA)' = kA'
(A+B)' = A' + B'
(AB)' = B'A' ← order reversed!

Example

If A = [2 8; 6 4]₂₂ then:
A' = [2 6; 8 4]₂₂
Rows become columns. Order changes from m×n → n×m.
⚠️

Reversal Rule (Critical): (AB)' = B'A' — the transpose of a product reverses the order. This is the #1 tested transpose property in boards and JEE.

🔁 Symmetric & Skew-Symmetric Matrices

Symmetric Matrix

A is symmetric if A' = A
i.e., aᵢⱼ = aⱼᵢ for all i, j
Example: [1 2 3; 2 5 4; 3 4 6]

Skew-Symmetric Matrix

A is skew-symmetric if A' = −A
i.e., aᵢⱼ = −aⱼᵢ for all i, j
All diagonal elements MUST be 0!
Key Theorem — Decomposition of Any Square Matrix
1
Any square matrix A can be written as: A = ½(A + A') + ½(A − A')
2
Let P = ½(A + A'). Then P' = ½(A + A')' = ½(A' + A) = P. So P is symmetric.
3
Let Q = ½(A − A'). Then Q' = ½(A − A')' = ½(A' − A) = −Q. So Q is skew-symmetric.
4
Therefore: A = P + Q = Symmetric + Skew-Symmetric ✓
💜 Board 3-Mark Proof — Must Know

Proving A = Symmetric + Skew-Symmetric is asked every year. Write P = ½(A+A'), show P'=P. Write Q = ½(A−A'), show Q'=−Q. Then verify A = P+Q. Show all 3 steps clearly.

⚙️ Elementary Operations (Transformations)
Used to find the inverse of a matrix via the method A = IA → I = A⁻¹A.

Three Row Operations

Rᵢ ↔ Rⱼ — Interchange rows i and j
Rᵢ → kRᵢ — Multiply row i by k (≠ 0)
Rᵢ → Rᵢ + kRⱼ — Add k×row j to row i

Three Column Operations

Cᵢ ↔ Cⱼ — Interchange cols i and j
Cᵢ → kCᵢ — Multiply col i by k (≠ 0)
Cᵢ → Cᵢ + kCⱼ — Add k×col j to col i
Finding Inverse by Elementary Row Operations
1
Write A = IA — place Identity matrix I on the right side
2
Apply the same row operations to both sides simultaneously to transform left-side A into I
3
When left side becomes I, the right side automatically becomes A⁻¹
4
Final: I = A⁻¹ · A, so right side = A⁻¹ ✓
⚠️

Critical Rule: Use ONLY row operations OR ONLY column operations in a single problem. Never mix both when finding inverse — it gives a wrong answer.

🔄 Invertible Matrix

Definition

A square matrix A is invertible if ∃ B such that AB = BA = I.
Then B = A⁻¹. Inverse is unique if it exists.

Reversal Rule

If A, B are invertible of same order:
(AB)⁻¹ = B⁻¹A⁻¹
Order is reversed — same as transpose!
⭐ Board & JEE Important Questions
Q1. If A and B are square matrices of equal order, which statement is always correct? (JEE 1995, 2M)
Hint: Test commutativity of addition vs multiplication. Only one property holds universally.
Answer: A + B = B + A (addition is always commutative). AB ≠ BA in general.
Q2. Express A = [2 7; 1 4] as sum of a symmetric and a skew-symmetric matrix. (Board 3M)
Hint: Compute A' first, then P = ½(A+A') and Q = ½(A−A'). Verify P' = P and Q' = −Q.
P = [2 4; 4 4] (symmetric), Q = [0 3; -3 0] (skew-symmetric). Check: P + Q = A ✓
Q3. If A = [1 2 2; 2 1 -2; a 2 b] satisfies AAᵀ = 9I, find the ordered pair (a, b). (JEE 2015 Main)
Hint: Each row must have magnitude 3 (row·row = 9). Use row 3 to get two equations in a and b.
Answer: (a, b) = (2, −1). Row 3: [a 2 b], a² + 4 + b² = 9 and 2a + 2 − 2b = 0.
Q4. For 3×3 matrices M and N, is NᵀMN symmetric or skew-symmetric when M is symmetric/skew-symmetric? (JEE 2013 Adv)
Hint: Take (NᵀMN)ᵀ = NᵀMᵀ(Nᵀ)ᵀ = NᵀMᵀN. Now substitute Mᵀ = M or Mᵀ = −M.
If M symmetric: (NᵀMN)ᵀ = NᵀMN (symmetric). If M skew: (NᵀMN)ᵀ = −NᵀMN (skew-symmetric). ✓
Q5. If M is 3×3, MᵀM = I and det(M) = 1, prove that det(M − I) = 0. (JEE 2004, 2M)
Hint: det(M−I) = det(M−MᵀM) = det(M)det(I−Mᵀ) = det(I−Mᵀ). Note det(Aᵀ) = det(A).
det(M−I) = 1 · det(I−Mᵀ) = det(I−M)ᵀ = det(I−M) = −det(M−I) ⇒ 2det(M−I) = 0 ✓
Q6. If A = [a b c; b c a; c a b] with a,b,c > 0, abc = 1 and AᵀA = I, find a³ + b³ + c³. (JEE 2003, 2M)
Hint: From AᵀA = I, get a²+b²+c² = 1 and ab+bc+ca = 0. Use a³+b³+c³−3abc = (a+b+c)((a+b+c)²−3(ab+bc+ca)).
Answer: a³ + b³ + c³ = 4.
💡 Must-Know Facts
🧠

Diagonal only in Square: You can discuss diagonal elements ONLY in a square matrix. Principal diagonal has exactly n elements (where i = j). Off-diagonal has (n²−n)/2 elements on each side.

⚠️

AB ≠ BA: Matrix multiplication is NOT commutative in general. This is the #1 exam mistake. Always double-check order when multiplying.

⚠️

AB = O ≠ A or B is zero: Unlike real numbers, if AB = O it does NOT mean A = O or B = O. Counter-examples exist.

🎯

Reversal Rule (Both!): (AB)' = B'A'  &  (AB)⁻¹ = B⁻¹A⁻¹. Both transpose and inverse reverse the multiplication order. Tested every year.

🎯

Skew-Symmetric diagonal = 0: Since aᵢᵢ = −aᵢᵢ ⇒ 2aᵢᵢ = 0 ⇒ aᵢᵢ = 0. All diagonal entries are zero. No exceptions.

📌

Determinant: Only defined for square matrices. For rectangular matrices, det is undefined.

🧠 Exam Strategy Tips
💜 Decomposition Proof (3 Marks)

Write P = ½(A+A') and Q = ½(A−A'). Show P' = P (symmetric). Show Q' = −Q (skew-symmetric). Then state A = P + Q. All 3 steps must appear for full marks.

💜 Matrix Multiplication — Check Order First

Always write: "A is m×n, B is n×p, so AB exists and is m×p." This 1-line check prevents errors and shows the examiner you know the condition.

💜 Finding Inverse — Be Consistent

Choose row ops OR column ops — not both. Write each step explicitly: R₂ → R₂ − 2R₁. Examiners award marks for each clearly written operation.

💜 Types Hierarchy to Remember

Identity ⊂ Scalar ⊂ Diagonal ⊂ Square. Each level adds a relaxed condition. Identity IS a scalar matrix. Scalar IS a diagonal matrix. Use this hierarchy to quickly classify any given matrix.

💜 Transpose of Product — JEE Trap

Students often write (AB)' = A'B'. This is WRONG. The correct answer is (AB)' = B'A'. Remember: the transpose of any product reverses the order — just like the inverse of a product.
